Let's Learn Some Syntaxis

Sintaxis en HTML

Nombre

Función

<DOCTYPE>

 Sirve para decirle al ordenador que el código que vamos a usar es HTML.

<Head>

 Contiene los metadatos de una página web.

<Title>

 Va dentro de <head> y es el título que le otorgamos a la página web, aparecera en la pestaña del navegador.

<Body>

 El cuerpo de la página.
